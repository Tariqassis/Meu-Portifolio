
# Meu Portifolio

web site Portifolio TariqDev



## Funcionalidades

- interação com a imagem do perfil com valina tilt
- animação de swipers ao descer 
- Responsivo para pc, tablet, celular
- Multiplataforma


## Stack utilizada

**Front-end:** HTML, CSS, JAVASCRIPT (AOS jS, Vanila Tilt)
